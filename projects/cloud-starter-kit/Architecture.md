# Cloud Starter Kit Architecture

## Objective

Deploy a reusable AWS environment that can host web applications securely using Docker.

## Infrastructure

- AWS VPC
- Public Subnet
- Internet Gateway
- Route Table
- Security Group
- Ubuntu EC2

## Services

- Docker
- Nginx
- Prometheus
- Grafana

## Future Improvements

- Private Subnet
- Load Balancer
- Auto Scaling
- RDS
- S3 Backups
